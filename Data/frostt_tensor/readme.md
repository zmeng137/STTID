# FROSTT Sparse Tensors

## Large Dataset Notice

For FROSTT sparse tensors that are too large for this GitHub repository, please download them directly from the official website:

**[FROSTT: Formidable Repository of Open Sparse Tensors and Tools](http://frostt.io/)**

The FROSTT website provides access to a comprehensive collection of real-world sparse tensor datasets for research and benchmarking purposes.

## Usage

After downloading the required tensors from the FROSTT website, place them in the appropriate directory as specified in the project documentation.
